Term Website Project: Planning Document

Description

Develop a planning document for the term website project, Salmon River Adventures, which describes the site purpose, target audience, site map, color scheme, typography and wireframe sketches for three screen sizes (small, medium, and large). 

Term Website Project Specifications

Site Purpose: Provide a brief statement about the purpose of the website drawing upon the client's functional specifications, the industry and any fictitious but viable goals that you deem to be reasonable.

Target Audience: Research who the website visitors will be by discovering the industry, visiting similar sites and asking your friends, family or acquaintances through an informal survey about the industry.  You may elect to develop personas for the distinct type of site visitors. This section should describe the target audience being judicious about the selection of the most plausible demographics.

Site Map: The site map is a sketch of all the website pages and references. It should illustrate the relationship between those pages. 

Color Scheme: The color scheme refers to the carefully selected site colors that reflect the organization's purpose and trademark colors. Note that Salmon River Adventures does not have established colors for brand identity. The planning document defines the color scheme through a color palette that labels color style definitions and swatches for site elements. 

Typography: The project specifications require that you utilized the directory of open source designer web fonts provided by fonts.google.com. The typography section of the planning document provides examples of the fonts to be used and where they are to be applied. 

Wire-frame Sketches: Produce wireframe sketches of your site design considering the purpose, specifications, and target audience. You should include sketches for small, medium, and large views respectively meaning ~smart phones, ~tablets and ~large width monitors.

Instructions

Save your completed planning document as a PDF file and post it as a new thread in the Planning Document Discussion Board for peer review. Return to the discussion board to review two peer submissions no later than Wednesday of Lesson 13. The review should contain constructive feedback on the readability, completeness, and relevance of each section required in the planning document. 

Assessment

The planning document and peer reviews associate with the discussion board are worth 25 points in the Term Project final grade category.
